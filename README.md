# alex-a

My own dedicated personal assistant, sadly not really meant for external use. 

I'm trying to keep it quite tight-on-lock so I can be sure it's safe to use. Eventually this will write my easy emails and give me LLM reminders so I don't need to rely on external services that I have to pay for!

## Setup

You need `uv` to run this. Run with:

```sh
uv run python main.py
```
